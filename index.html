웹 퍼블리셔 최원빈입니다.

매일매일 성장하고 있는 웹 퍼블리셔 전혜린입니다. "성실함을 이기는 것은 없다"는 마음가짐으로 기술 블로그에 공부 기록을 꾸준히 하여 새롭게 배운 내용은 내 것으로 만들고자 노력하고 있습니다. 또한 협업에 필요한 깃허브의 Git 명령어를 학습하였으며 항상 커밋하는 습관을 들여 약 5개월간 1,000건 이상의 commit 경험이 있습니다. 저에게 퍼블리싱은 제가 하고 싶은 일이 무엇인지 깨닫게 해준 직무입니다. 이 열정으로 꾸준히 노력하여 늘 발전하는 웹 퍼블리셔가 되겠습니다.
